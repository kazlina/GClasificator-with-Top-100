Инсрукция по внедрению скрипта классификации в систему.

1. Открыть workbench, зайти в БД системы (предположительно test).
2. Нажать правой кнопкой мыши на Routines, выбрать Create Routine...
3. Открылась штука для создания процедуры.
4. Вместо new_routine написать название процедуры, например "classification" (всего процедур 6, 1 главная и 5 неглавных, названия написаны в файле test_routines.sql)
5. Теперь нужно добавить тело процедуры. 
	5.1. Скопировать тело процедуры из файла test_routines.sql
	5.2. Вставить то, что скопировали.
6. Сохранить: нажать Apply. 
6. Повторить это действия 2-6 для всех процедур.
