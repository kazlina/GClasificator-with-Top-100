Инсрукция по внедрению скрипта классификации в систему.

1. Открыть workbench.
2. Сделать импорт файла test_routines.sql
3. Выполнить скрипт в workbench (вместо group_id подставить любое id из талицы groupdescr, например, 2)
	USE test;
	CALL classification( group_id );
4. Подождать. (минута, час, день)
5. Написать Юльке, работает или нет. Как долго работает.
6. Можно запустить скрипт много раз.
